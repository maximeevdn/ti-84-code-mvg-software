## Installation Guide
1. **Download:** Get the `MVGOS.8xp` file from this repository.
2. **Connect:** Plug your TI-84 Plus CE into your computer via USB.
3. **Software:** Open **TI Connect™ CE**.
4. **Transfer:** Drag `MVGOS.8xp` into the "Calculator Explorer" tab.
5. **Launch:** Press `prgm` on your calculator, select `MVGOS`, and hit `ENTER`.
